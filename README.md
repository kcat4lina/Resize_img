# Img Resizing Tool

This is a free to use tool for resizing images.

## Features

It supports *.png 
It supports *.jpg
It supports *.bmp
It supports *.jpeg
It supports *.gif

## Getting Started

1. Clone this repository
2. Run the dist/resize.exe
3. Choose image to resize
4. Download the resized image
